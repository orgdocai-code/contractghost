# ContractGhost Manual Review Request Tracker

> Run 36 operational asset
> Purpose: capture every founder-led manual review request in one place so early demand turns into structured learning, not scattered email.

## How To Use
For every inbound manual review request:
1. Add a new row immediately when the request arrives
2. Update status as it moves through triage → review → sent → follow-up
3. Copy useful user language into the Notes / Signal column
4. After reply, mirror key learnings into `memory/contractghost-feedback-tracker.md`

## Status Definitions
- `NEW` — request arrived, not triaged yet
- `NEEDS_INFO` — missing file / clause / context
- `IN_REVIEW` — contract is being reviewed now
- `SENT` — manual review delivered
- `FOLLOW_UP` — waiting on user response after delivery
- `CLOSED` — complete, no current action
- `BAD_FIT` — outside target scope / needs lawyer

## Signal Prompts
When filling `Outcome / Signal` or `Notes`, look for:
- What clause scared them most?
- Did they say they were about to sign today?
- Did they compare this to ChatGPT, templates, or a lawyer?
- Did they mention money saved, confidence gained, or negotiation leverage?
- Did they hint at one-off vs recurring need?
- Did they react positively or negatively to pricing?

## Fast Triage Rules
### Good fit
- Routine freelance client contract
- SOW / MSA / subcontractor agreement
- Website, design, marketing, copywriting, consulting work
- Clear concern around scope, payment, IP, termination, liability

### Borderline / ask one question
- Missing attachment
- Huge agreement but only one clause matters
- Urgency is high but context is thin

### Bad fit / de-prioritize
- Employment contracts
- Equity / acquisition / litigation-heavy docs
- Regulated / enterprise-heavy legal paper
- Formal legal-advice expectations

## Follow-Up Questions Library
Use only one when needed:
- "Can you send the specific clause or attach the agreement PDF?"
- "What part worries you most: scope, payment, IP, termination, or liability?"
- "Is this something you need to sign today, or are you comparing options first?"
- "If we flag the top issues fast, would you want suggested rewrite wording too?"

## Weekly Review Prompt
At the end of each week, review the tracker for:
- most common contract type
- most common pain point
- strongest willingness-to-pay signal
- strongest trust objection
- whether the wedge should narrow further
