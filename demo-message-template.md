# ContractGhost Demo Message Template

## Short version
Thanks — here's the quick demo of what I'm building.

ContractGhost checks freelancer contracts before signing and flags risky bits like vague scope, unlimited revisions, one-sided termination, IP grabs, and bad payment terms.

The basic flow is:
1. Upload contract
2. Get a risk summary
3. See plain-English explanations + suggested pushback language

Main question: would something like this have helped before you signed your last contract?

The point isn't "AI legal advice" — it's fast risk spotting and negotiation prep so freelancers can catch obvious red flags before they get trapped.
