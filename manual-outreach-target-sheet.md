# ContractGhost Manual Outreach Target Sheet

## Context
Created on 2026-03-14 during founder-loop run 26. The process now forbids more vague prep. This file turns the first-10 plan into exact conversation slots so the next human-available execution window can be used without thinking.

## Goal
Have a fixed list of 10 conversation shots, each with persona, trigger, message angle, and evidence target.

## Message-angle mapping
- **pain-first:** use when the person already expressed pain publicly
- **workaround-first:** use when they seem pragmatic and process-driven
- **ChatGPT-objection path:** use when they are already AI-comfortable
- **screenshot intro:** use when a short visual demo would lower friction

## Minimum evidence standard
A slot only counts when at least one of the following is captured:
- a specific clause or contract failure
- the current workaround they rely on
- a trust objection to AI/legal tooling
- a beta ask or price reaction

## Anti-loop guardrail
If no public URL exists next run, do not make another targeting document. Use this exact sheet plus the existing screenshot demo path and log the first real response in `contractghost/beta-intake-log.md`.
